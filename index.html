📥 Installation Instructions

Method 1: Direct Repository Source (Recommended)

  1. In Kodi File Manager, add source:
    https://classymouse.github.io/repository.thecrew.alpha/
    Name it "crew-development" and click OK
  2. Go to: Settings → Add-ons → Install from repository → The Crew Alpha Repository
  3. Install the addons you want to test

Method 2: Repository ZIP Download

  1. Download the repository installer:
    https://classymouse.github.io/repository.thecrew.alpha/repository.thecrew.alpha-1.0.7.zip
    Download Repository Installer
  2. In Kodi, go to: Settings → Add-ons → Install from zip file
  3. Navigate to the downloaded file and install repository.thecrew.alpha-1.0.7.zip
  4. Wait for the "The Crew Alpha Repository Add-on installed" notification
  5. Go to: Settings → Add-ons → Install from repository → The Crew Alpha Repository
  6. Install the addons you want to test

🚀 Orion Setup (Optional - Enhanced Scraping)

Orion enhances The Crew with premium torrent, usenet, and hoster links. Free tier includes 100 daily searches.

Step 1: Create Orion Account

  1. Visit https://orionoid.com and create a free account
  2. After registration, log in to your account
  3. Go to the Settings → Account section
  4. Copy your API Key (keep this safe!)

Step 2: Configure in The Crew

  1. In Kodi, go to The Crew → Tools → Orion Settings
  2. Enter your Orion API Key
  3. Enable Orion integration
  4. Configure your preferred filters (quality, size, providers, etc.)
  5. Save settings and enjoy enhanced scraping!

Tip: Free accounts get 100 searches/day. Premium plans offer unlimited searches, faster speeds, and priority support. Visit orionoid.com for upgrade options.
